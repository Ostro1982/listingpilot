1. Authentication

Every API request needs a header with your personal API key:

Authorization: Bearer {{ agent.api_key }}

Your API key

{{ agent.api_key }}
Keep it secret. Anyone with this key can create listings on your behalf and use your credits. Regenerate it if it leaks.

2. Try it now

Paste JSON below and click "Run". This sends a real request to your account (uses 1 credit if creating).

3. Create listing

POST/api/listings

Body fields

  • property.property_type — Single Family / Condo / Townhouse / Multi-Family / Land / Apartment
  • property.operation — Sale / Rent
  • property.address, city, state (2-letter), zip_code
  • property.price (integer USD), bedrooms, bathrooms, living_area_sqft
  • property.features — array of strings
  • photos — array of URLs (https://...) or base64 data URIs
  • auto_publish — boolean (default false). If true, posts to {{ social_platforms }} via Upload Post.
curl -X POST {{ public_base }}/api/listings \
  -H "Authorization: Bearer {{ agent.api_key }}" \
  -H "Content-Type: application/json" \
  -d '{
    "auto_publish": true,
    "property": {
      "property_type": "Single Family",
      "operation": "Sale",
      "address": "1234 Maple Ave",
      "city": "Austin",
      "state": "TX",

4. List your listings

curl {{ public_base }}/api/listings \
  -H "Authorization: Bearer {{ agent.api_key }}"

5. Update listing

PATCH/api/listings/<listing_id>

Common patches

# Drop the price + regenerate text
curl -X PATCH {{ public_base }}/api/listings/abc123 \
  -H "Authorization: Bearer {{ agent.api_key }}" \
  -H "Content-Type: application/json" \
  -d '{"property":{"price":425000},"regenerate_text":true}'

# Just override the headline manually
curl -X PATCH {{ public_base }}/api/listings/abc123 \
  -H "Authorization: Bearer {{ agent.api_key }}" \
  -d '{"headline":"PRICE REDUCED — Open House Sunday"}'

# Add more photos
curl -X PATCH {{ public_base }}/api/listings/abc123 \
  -H "Authorization: Bearer {{ agent.api_key }}" \
  -d '{"photos_add":["https://example.com/photo3.jpg"]}'

# Re-publish to social
curl -X PATCH {{ public_base }}/api/listings/abc123 \
  -H "Authorization: Bearer {{ agent.api_key }}" \
  -d '{"republish":true}'

6. Delete listing

DELETE/api/listings/<listing_id>
curl -X DELETE {{ public_base }}/api/listings/abc123 \
  -H "Authorization: Bearer {{ agent.api_key }}"

7. Tools & ideas

  • n8n — visual workflow builder. Trigger: Google Sheets new row → HTTP POST to /api/listings → done.
  • Zapier / Make — same but cloud-based.
  • Excel/CSV bulk import — Python script reads file → loops → POST per row.
  • Form on your own site — embed a custom form anywhere → JS sends to /api/listings.
  • MLS sync — scheduled cron pulls new listings from your MLS feed → posts here automatically.
